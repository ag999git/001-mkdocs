## Chapter 5 Functions Part I
### Python Questions and Answers

#### a. A function is a “named sequence of statement(s)” that performs a computation. Explain.
A **function** groups a set of statements under a **name** so we can reuse them.  
Example:
```python
def add(a, b):
    return a + b
```
Here, `add` is the name, and the statements inside perform a computation.

#### b. Why do we use “functions” in programming?
- To avoid repetition  
- To organize code better  
- To improve readability  
- To break large programs into small manageable units  
- To promote reusability  

#### c. Every Python function returns a value; if not specified, it returns `None`. Explain.
When Python reaches the end of a function without encountering `return`, it automatically returns `None`.
```python
# The given function hello() does not have a return statement
def hello():
    print("Hi")

x = hello()    # hello() returns a None
print(x)  # None
```

#### d. If two objects have same id, they point to the same object. But two equal objects may not have same id. Explain.
- Same `id` → same memory location → guaranteed identical object.  
- Equal (`==`) objects → values match but memory locations may be different.
```python
a = [1,2]
b = [1,2]
print(a == b)  # True
print(id(a) == id(b))  # False
```

#### e. Two meanings of `str()` function.
1. **Casting**: Convert data to string datatype.  
   ```python
   str(123)  # "123"
   ```
2. **String representation of an object** (human-readable form).  
   ```python
   str([1,2,3])  # "[1, 2, 3]"
   ```

#### f. Meaning of semi-open interval [0.0, 1.0)
Includes 0.0 but **excludes** 1.0.  
Range: **0.0 ≤ x < 1.0**

#### g. Why initialize a random number generator? What does hashable seed mean?
- Seeding ensures **predictable**, **repeatable** randomness (useful for testing).  
- A seed must be **hashable**, meaning it can be used as a dictionary key.  
  Hashable types: int, float, str, tuple.

#### h. Why random numbers are called pseudo-random?
They are **generated by algorithms**, not truly random.  
They *appear* random but are fully determined by the seed.

#### i. Difference between a function and a method.
- **Function**: Independent block of code.
- **Method**: A function that belongs to an object/class.
```python
len("hi")      # function
"hi".upper()   # method
```

#### j. A function must be defined before use. Why?
Python executes code **top to bottom**.  
It must know a function’s definition before executing a call.

#### k. What is the lifetime of a variable?
The period during which a variable **exists in memory**.  
- Local variables → exist during function execution  
- Global variables → exist as long as program runs  

#### l. Naming conventions for functions in Python.
- Use **snake_case** → `calculate_total()`  
- Use meaningful names  
- Start with a verb (usually action)  
- Avoid uppercase (reserved for classes)

#### a. Which functions convert a character into ASCII and vice-versa?

**Answer:**  
- `ord(char)` → converts a character to its ASCII code  
- `chr(number)` → converts ASCII code to a character  

```python
print(ord('A'))  # 65
print(chr(65))   # 'A'
```

#### b. Script: Print all numbers from 1 to 100 where all digits are even  
(Even digits = 0, 2, 4, 6, 8)

```python
for num in range(1, 101):
    digits = str(num)
    if all(int(d) % 2 == 0 for d in digits):
        print(num)
```

#### f. Determine odd/even by flipping a boolean flag n times

```python
n = int(input("Enter a number: "))
flag = True  # initial state

for _ in range(n):
    flag = not flag  # flip
if flag == True:
    print(n, "is even")
else:
    print(n, "is odd")
```

#### g. Script with function count_vowel() that counts vowels in a string

```python
def count_vowel(s):
    vowels = "aeiouAEIOU"
    return sum(1 for ch in s if ch in vowels)

text = input("Enter a string: ")
print("Number of vowels:", count_vowel(text))
```

#### i. Coin toss simulation for n = 100 and n = 500

```python
import random

def simulate(n):
    options = ['Head', 'Tail']
    counts = {'Head': 0, 'Tail': 0}

    for _ in range(n):
        toss = random.choice(options)
        counts[toss] += 1

    print(f"For n = {n}:")
    print("Heads:", counts['Head'], "Probability =", counts['Head']/n)
    print("Tails:", counts['Tail'], "Probability =", counts['Tail']/n)
    print()

simulate(100)
simulate(500)
```
